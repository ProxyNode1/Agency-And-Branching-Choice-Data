## Using the files
LimeSurvey can import txt or lss files to create the survey.

## Adding Videos
Once the survey is created, copy the survey's id. Go to "upload/survey" folder inside the LimeSurvey's folder. Create a new folder and change its name to match the survey's id. Copy the "files" folder inside the Dusk Vid.7z, and paste it inside the survey's folder. Previewing any of the Agency_Choice_Ext question group's details can be used to check if the videos are in the right location and is seen in the survey.

## Choice Selection Menu on the "Break" page
Cr_3 = NBD_LM (1, 2, 3 <- applicable values for the menu)
Mo_1_2 = NB_LM (1, 2)
Mo_Bed_2 = B_HM (1, 2)
Hos_DvD = NB_HM (1, 2)
Hos_DvJ = B_LM (1, 2)
Hos_DW_Pnk = NBD_HM (1, 2, 3)
Hos_AimGun = did the player selected "Drop It" or "Point It", when in the episode1, the player character is asked to take away the gun from the armed captors. (1, 2)
Hos_DogCalm = did the dog calm down during B_LM. (1 = No, 2 = Yes, 3 = Did not happen)
Hos_DogKc = was the dog kicked during B_LM. (1 = No, 2 = Yes, 3 = Did not happen)
